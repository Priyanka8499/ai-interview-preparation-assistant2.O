# Requirements File

## Project Name
AI Interview Preparation Assistant

## Project Objective
The AI Interview Preparation Assistant helps users prepare for interviews by generating AI-powered interview questions and sample answers based on the selected job role, interview type, and difficulty level.


## Functional Requirements

- Users can enter a job role.
- Users can select the interview difficulty level.
- Users can choose the interview type (Technical, HR, Behavioral, or Mixed).
- The application generates interview questions using the Google Gemini API.
- The application generates sample answers for each interview question.
- Users can view the generated questions and answers in a clean and responsive interface.


## Non-Functional Requirements

- Responsive and user-friendly interface.
- Fast response time for AI-generated content.
- Secure API key management using environment variables.
- Cross-browser compatibility.
- Easy deployment using Vercel.

## Software Requirements

- npm
- Visual Studio Code
- Git
- GitHub
- Google Chrome (recommended)

## Technologies Used

- React.js
- TypeScript
- Vite
- CSS
- Google Gemini API
- Git & GitHub

## Hardware Requirements

### Minimum
- Dual-core processor
- 4 GB RAM
- Stable internet connection

### Recommended
- Intel Core i3/i5 or equivalent
- 8 GB RAM
- Modern web browser


## Conclusion

The AI Interview Preparation Assistant is a web-based application that provides personalized interview preparation using Google's Gemini AI. It offers an easy-to-use interface, dynamic question generation, and sample answers to help users improve their interview skills.
